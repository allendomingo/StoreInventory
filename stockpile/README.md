# Stockpile Component Library
Component library for use in the StockFlow inventory web app.

## Usage
### Setup
1. From the root of the repo, `cd stockpile`
2. Run `npm install` or `yarn install`

### Run storybook
`npm run storybook` or `yarn run storybook`
